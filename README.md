An immersive high-tech 3D isometric presentation built with React, Vite, and Tailwind CSS.

## ⚠️ STOP HERE! READ THIS TO FIX THE BLACK SCREEN ⚠️

If you see a black screen on your website, it's because **GitHub doesn't know how to run the source code yet.** You MUST tell GitHub to "Build" the project using the "GitHub Actions" I have added.

### Step-by-Step Fix:

1.  **Push these changes** to your GitHub repository.
2.  **Go to GitHub.com** and open your repository.
3.  **Click on Settings** (at the top).
4.  **Click on Actions -> General** (on the left side).
    *   Find **"Workflow permissions"** (at the bottom).
    *   Select **"Read and write permissions"**.
    *   Click **Save**.
5.  **Click on Pages** (on the left side).
    *   Look for **"Build and deployment"** -> **"Source"**.
    *   Change it from "Deploy from a branch" to **"GitHub Actions"**.
6.  **Check the top "Actions" tab**.
    *   You should see a yellow circle (or green checkmark) for "Deploy to GitHub Pages".
    *   Wait for it to finish.

**Once the yellow circle turns into a green checkmark, your website will work!**
